FAQ

Internet of Things (IoT) ou Internet das Coisas é o conceito de conectar qualquer dispositivo à internet e outros dispositivos conectados. IoT é uma rede gigante de coisas conectadas - todas coletando e compartilhando dados sobre como são usadas e o ambiente ao seu redor.

Utilizamos diversos sensores como: sensor de temperatura DHT22, sensor de umidade, sensor ultrassônico HC-SR04 para medição de distância, sensor de velocidade, módulo GPS, sensor de luminosidade LDR, sensor de movimento PIR, e sensores de pressão.

Os sensores coletam dados em tempo real e os enviam via WiFi para nosso sistema. Os dados são processados e armazenados em banco de dados, depois exibidos em gráficos interativos neste portal para análise e visualização.

Utilizamos principalmente C++ para Arduino, Python para processamento de dados, JavaScript para interface web, HTML/CSS para frontend, e SQL para banco de dados. Cada linguagem tem seu papel específico no sistema.

Entre em contato através da página de contatos! Estamos sempre buscando novos membros interessados em robótica e IoT. Não é necessário experiência prévia, apenas vontade de aprender e colaborar.

Recomendamos começar com Arduino, cursos online gratuitos, documentação oficial, e participar de comunidades como fóruns e grupos de estudo. Nossa seção de vídeos também oferece tutoriais práticos.
