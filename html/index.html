Sobre o Projeto

Este portal foi desenvolvido como parte da disciplina de IoT (Internet das Coisas), documentando nossas experiências práticas com robôs e sensores para coleta de dados como temperatura, umidade e velocidade.

O objetivo é criar um repositório multimídia das atividades realizadas em laboratório e um painel interativo para visualização dos dados coletados pelos sensores.
